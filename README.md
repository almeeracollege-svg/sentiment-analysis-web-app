# Sentiment Analysis Web App

A web application built using Flask and TextBlob that analyzes user-entered text and determines whether the sentiment is Positive, Negative, or Neutral.

## Features

- Sentiment Classification
- Polarity Score
- Sentiment Interpretation
- User-Friendly Interface

## Technologies Used

- Python
- Flask
- TextBlob
- HTML
- CSS

## Run the Project

Install dependencies:

pip install flask textblob

Run:

python app.py

Open:

http://127.0.0.1:5000
